# 志愿者管理系统 - 功能设计文档

## 1. 核心功能模块

### 1.1 地图可视化模块（核心）

*   **完整世界地图**
    *   包含所有国家边界。

*   **快速聚焦栏（地图上方固定区域）**
    ```
    [中国大陆] [中国台湾] [东南亚] [美国] [欧洲] [重置世界视图]
    ```
    *   点击按钮自动缩放/聚焦到对应区域。
    *   **中国大陆**：显示省级边界。
    *   **其他区域**：显示国家边界。

*   **交互方式**
    *   **点击**国家/省份：打开该地区志愿者面板。
    *   **鼠标悬停**：显示地区名称和志愿者数量。

## 8. 交互流程总结

1.  **用户进入网站** → 看到完整世界地图 + 顶部统计 + 底部筛选器。
2.  **想要查看某区域** →
    *   **方式A**：点击顶部快速聚焦栏按钮。
    *   **方式B**：直接点击地图上的国家/省份。
3.  **地图聚焦到区域** → 底部显示该地区志愿者面板。
4.  **筛选志愿者** → 使用底部筛选器工具栏。
5.  **查看志愿者详情** → 点击志愿者卡片。
6.  **查看/添加服务记录** → 在个人面板中操作。

## 9. 关键特性强调

### 9.1 地图交互优先级
*   快速聚焦栏提供一键导航。
*   支持从世界视图直接点击。
*   保持地图始终在视觉中心。

### 9.2 数据展示优化
*   默认显示4张卡片，避免信息过载。
*   提供“展开完整列表”选项。
*   分页加载大量数据。

### 9.3 简化服务记录系统
*   取消评分和标签系统。
*   专注于事实记录：类型、时长、描述。
*   保留审核状态机制（管理员用）。

### 9.4 响应式布局考虑
*   **桌面端（宽屏）**：上-中-下三部分。
*   **平板端**：地图区域扩大，面板可折叠。
*   **移动端**：地图全屏，点击后弹出面板。
