# 纯金经典翻译计划义工追踪系统

## 🌟 项目简介
为纯金经典翻译计划开发的全球义工管理与追踪平台，提供可视化地图界面和义工数据管理功能。

## 🗺️ 主要功能
- 全球地图可视化展示义工分布
- 国家/地区点击查看详细信息
- 中国（省市）和美国（州）二级区域划分
- 义工信息的增删改查
- 数据统计与导出功能

## 🚀 快速开始
1. 克隆仓库：`git clone https://github.com/puregold-classic/volunteer-tracker.git`
2. 打开 `index.html` 在浏览器中运行
3. 或访问在线版本：https://puregold-classic.github.io/volunteer-tracker/
